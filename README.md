# 📋 Student Attendance Tracker

## 📌 CSE2006- Programming in JAVA Project (VITyarthi)

A simple Java program to manage student attendance. Built as a college project to learn object-oriented programming and file handling.

SAMRIDHI SHREE 24BAI10161

## ✨ Features

- ➕ Add new students
- 👀 View all students
- ✅ Mark daily attendance (yes/no)
- 📅 View attendance reports by date
- 📊 See student-wise attendance percentage
- 💾 Data saves automatically to files

## 🚀 How to Run

1. Compile the program:
2. Run it: javac AttendanceTracker.java


## 🎮 How to Use

| Option | What it does |
|--------|---------------|
| 1 ➕ | Add a student (name + registration number) |
| 2 👀 | Show all students |
| 3 ✅ | Mark attendance for today |
| 4 📊 | View reports (by date, all records, or student summary) |
| 5 💾 | Save and exit |

## 📁 Files Created

- `students.txt` – Stores student details 👨‍🎓
- `attendance.txt` – Stores attendance records 📝

## 🛠️ Requirements

- ☕ Java 8 or higher
- 💻 Any operating system (Windows, Mac, Linux)
